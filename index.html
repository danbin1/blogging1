랜덤 로또 번호

1. 랜덤한 수를 담을 수 있는 배열을 만들어준다.

2. 랜덤한 수를 몇 개 담을 것인지 count 변수에 표현한다.

3. count 변수에 담긴 횟수만큼 for문을 만들어 돌린다.

4. for문 안에서 1 ~ 45까지의 랜덤한 번호를 randomNum 변수에 담는다.

5. 만약 randomNum 번호가 randomLotto 배열 안에 있는지

indexOf를 사용하여 결과 값이 -1이 나오면 없는 걸로 간주

randomLotto 배열 안에 추가.

6. 그렇지 않고 -1이 아니면 randomNum 번호가

randomLotto 배열 안에 있다는 걸로 간주하고 i-- i 변수의 -1를 해

for문을 다시 돌게 한다.

7. 마지막으로 sort()함수를 사용하여 오름차순 정렬을 해주고

randomLotto 배열을 리턴한다.
